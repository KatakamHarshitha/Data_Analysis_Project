# Iris Data Analysis Project

## Overview
This project performs basic data analysis on the Iris dataset using Python.

## Features
- Load CSV data using Pandas
- Display the first five rows
- Show dataset information
- Generate summary statistics
- Calculate average sepal length
- Create:
  - Bar Chart
  - Scatter Plot
  - Correlation Heatmap

## Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
